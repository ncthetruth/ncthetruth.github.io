Tanaka Guitar

Tanaka Guitar E-commerce Project

What is Tanaka Guitar?

The Tanaka Guitar project is a university assignment where we are required to create an online store selling guitars. We are only allowed to use pure HTML, CSS, and JS without utilizing any frameworks. This project aims to develop our skills in web development and showcase our ability to create a functional and visually appealing e-commerce platform within the given constraints. The focus is on implementing a seamless user experience, efficient design, and interactive features using fundamental web technologies.

Wireframe And Prototype

The Tanaka Guitar project requires the development of a user-friendly website with an emphasis on an intuitive and easy-to-navigate interface, ensuring a positive user experience. The design approach prioritizes accessibility and enjoyment for visitors to the online guitar store, promoting usability and engagement. Our goal is to enhance overall user satisfaction through thoughtful design choices and user-centric features.
